Detailed Description

template<typename T, std::size_t N, std::uintmax_t polynomial>
struct crc3x::generator< T, N, polynomial >

A structure that generates CRC tables.

The CRC table values that are produced with this are all compile-time constant (i.e. constexpr). This has the advantage that there is no runtime overhead cost to generating the tables regardless of the CRC algorithm used.

The generator is typically not used directly by applications but by instantiating Crc.
